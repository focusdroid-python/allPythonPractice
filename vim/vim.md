###插入方式
    -i : 在光标前一个插入
    -I : 在行首插入
    -A : 在行末插入
    -a : 在光标后一个插入
    -o : 在下一行插入
    -O : 在上一行插入

###保存
    - ESC --> :wq:保存并退出
    -
###复制
    - yy : 复制光标所在一行
    - 4yy: 复制光标所在行以下四行
    
    - p :  粘贴光标所在行
    - dd:  删除（剪切）光标所在行
    - 4dd: 删除（剪切）光标所在行以下四行
    
    - j : 上一行
    - k : 下一行
    - l : 右一行
    - h : 左一行
    
    - ctrl + b：往上翻一页
    - ctrl + f：往下翻一页
    
    - H: 当前屏幕上方
    - M: 当前屏幕中间
    - L: 当前屏幕下方
    
    - 20G : 快速定位第20行
    - G : 快速回到整个代码最后一行
    - gg : 快速回到整个代码第一行
    - w : 向后跳一个单词的长度
    - b ： 向前跳一个单词的长度
